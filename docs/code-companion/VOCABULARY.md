# 词汇表

> 遇到新术语自动追加。不用背，忘了来查。

| 术语 | 人话 | 在哪见过 |
|------|------|----------|
| `{Binding XXX}` | 界面的值从代码里拿，代码变了界面跟着变 | MainWindow.xaml |
| `{StaticResource XXX}` | 引用预定义的值（比如颜色名） | MainWindow.xaml |
| `Width="*"` | 占剩余所有空间 | MainWindow.xaml |
| `Margin="10,5,10,5"` | 上右下左的留白（像素） | MainWindow.xaml |
| `Content="XXX"` | 按钮上显示的文字 | MainWindow.xaml |
| `[ObservableProperty]` | 标记属性，让界面能跟着它变化 | ViewModel |
| `[RelayCommand]` | 标记方法，让按钮能触发它 | ViewModel |
