# PRD - ZTD (Zen To Done)

> 创建时间: 2026-02-27
> 最后更新: 2026-02-27

---

## 核心价值

帮助个人通过简单专注的想法收集和任务管理系统，提升生产力和实现目标。

---

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| Windows | C# + WinUI 3 | 微软官方桌面框架 |
| 数据库 | SQLite | 本地持久化存储 |
| 数据访问 | Microsoft.Data.Sqlite | 微软官方，成本最低 |
| MVVM 支持 | WinUI 3 原生数据绑定 | 简化 UI 代码 |
| 依赖注入 | Microsoft.Extensions.DependencyInjection | .NET 内置 |
| UI风格 | Google Material Design | 简洁直观 |
| 后期扩展 | Kotlin + Jetpack Compose | Android 原生开发（后续）|

### 分层架构（MVVM）

```
┌─────────────────────────────────────────────────┐
│              Views (WinUI 3 界面)                 │
│         - XAML 定义 UI                            │
│         - 数据绑定到 ViewModel                     │
└──────────────────┬──────────────────────────────┘
                   │ 数据绑定 / 命令
┌──────────────────▼──────────────────────────────┐
│         ViewModels (视图模型)                    │
│         - 业务状态管理                            │
│         - 命令处理                                │
│         - 调用 Services                            │
└──────────────────┬──────────────────────────────┘
                   │ 调用
┌──────────────────▼──────────────────────────────┐
│           Services (业务逻辑)                     │
│         - 业务规则处理                             │
│         - 调用 Repositories                       │
└──────────────────┬──────────────────────────────┘
                   │ 调用
┌──────────────────▼──────────────────────────────┐
│          Data (数据访问层)                         │
│         - Repository 模式                         │
│         - SQLite CRUD 操作                         │
└──────────────────┬──────────────────────────────┘
                   │
┌──────────────────▼──────────────────────────────┐
│          SQLite 数据库                          │
└───────────────────────────────────────────────┘
```

### 数据流转

```
用户操作 (Views)
  ↓
ViewModel 处理命令
  ↓
Service 处理业务逻辑
  ↓
Repository 操作数据库
  ↓
SQLite 持久化

数据反向同步通过数据绑定自动更新 UI
```

### 技术选型说明

| 技术 | 选择理由 |
|------|----------|
| MVVM | WinUI 3 官方推荐模式，UI 与逻辑分离 |
| Microsoft.Data.Sqlite | 微软官方，零依赖，成本最低 |
| Repository 模式 | 数据访问与业务逻辑分离，易于测试 |
| 依赖注入 | 解耦，便于单元测试和后期扩展 |
| WinUI 3 数据绑定 | 减少样板代码，提升开发效率 |

## 数据模型

### Idea（想法）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| content | string | 想法内容 |
| createdAt | timestamp | 创建时间 |

### Task（任务）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| content | string | 任务内容 |
| status | enum | todo / today / done |
| timeTag | string | <30分钟 / >30分钟 |
| categoryTag | string | 工作 / 个人 / 学习 / 运动 / ... |
| createdAt | timestamp | 创建时间 |
| completedAt | timestamp | 完成时间 |

### Note（笔记）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| content | string | 笔记内容 |
| category | string | 分类标签 |
| createdAt | timestamp | 创建时间 |
| updatedAt | timestamp | 更新时间 |

### Tomato（番茄钟记录）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| taskId | string | 关联任务ID |
| duration | int | 实际时长（秒），>10秒才记录 |
| targetDuration | int | 目标时长（秒） |
| startTime | timestamp | 开始时间 |
| completedAt | timestamp | 完成时间 |

### CompletedTask（已完成任务）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| taskContent | string | 任务内容 |
| completedAt | timestamp | 完成时间 |

## 业务流程

### 想法流转

```
想法收集
  ├─ 删除
  ├─ 待办 → 待办列表（打标签：时间+分类）
  ├─ 委派（后续功能）
  ├─ 笔记 → 笔记库
  └─ 快速完成 → 归档统计
```

### 任务流转

```
待办列表
  ├─ 小于30分钟 → 批量勾选 → 归档统计
  └─ 大于30分钟 → 选择加入今日待办（最多3个，可替换）
                      ↓
                 今日待办
                      ↓ (开启番茄钟)
                 归档统计
```

### 番茄钟流程

```
点击番茄钟按钮
  ↓
自定义时长（或使用默认）
  ↓
开始计时
  ├─ <10秒 → 可撤回，不记录
  └─ >10秒 → 记录到番茄钟统计
  ↓
时间到 → 自动锁定屏幕
  ↓
用户点击休息 → 开始休息计时
```
